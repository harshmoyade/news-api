News Backend APIs

Step 1 => After cloning Run this composer cmd 
        **composer install**

Step 2 => Create Database using name "**news**"

Step 3 => Run Artisan CMD for Migration "**php artisan migrate**"

Step 4 => Run "**php artisan news:fetch**"  cmd for fetch latest news from "Nyt, Guardian, News API source"
